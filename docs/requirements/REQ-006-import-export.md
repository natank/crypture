# REQ-006: Import/Export

## Related Backlog Items
- [10] Export Portfolio to CSV/JSON
- [11] Import Portfolio from CSV/JSON

## Description
Users should be able to export and import their portfolio data.

## Functional Requirements
1. Users shall be able to export their portfolio to CSV and JSON.
2. Users shall be able to import their portfolio from CSV and JSON.

## Non-Functional Requirements
- The import process should validate the file format.
