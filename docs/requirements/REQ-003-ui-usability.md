## Description
The application must provide a clean, intuitive, and responsive user interface.

## Functional Requirements
1. The application shall display a portfolio overview.
2. Users shall be able to filter and sort assets.
3. The application shall provide visual feedback for loading and error states.
4. The application shall have a landing page.

## Non-Functional Requirements
- The UI must be responsive.
- The UI must be accessible.
