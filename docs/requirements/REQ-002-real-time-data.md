# REQ-002: Real-Time Data Integration

## Related Backlog Items
- [4] Fetch Real-Time Prices via CoinGecko API
- [5] Real-Time Value Updates on Change

## Description
The application needs to fetch real-time prices to provide accurate portfolio valuation.

## Functional Requirements
1. The application shall fetch current prices from the CoinGecko API.
2. Prices shall be updated automatically when the portfolio changes.
3. The application shall handle API errors gracefully.

## Non-Functional Requirements
- API usage should be optimized to avoid rate limits.
