# REQ-001: Core Portfolio Management

## Related Backlog Items
- [1] Add Crypto Asset with Quantity
- [2] Delete Crypto Asset
- [3] Calculate and Display Total Portfolio Value
- [16] Edit Asset Quantity

## Description
The core functionality of the application is to allow users to manage their crypto portfolio. This includes adding assets, removing them, and seeing the total value.

## Functional Requirements
1. Users shall be able to add a crypto asset by selecting it from a list and entering a quantity.
2. Users shall be able to remove an asset from their portfolio.
3. Users shall be able to edit the quantity of an existing asset.
4. The application shall calculate the total value of the portfolio based on current prices.

## Non-Functional Requirements
- The portfolio state must be persisted (see REQ-005).
- Operations should be instant from a user perspective.
