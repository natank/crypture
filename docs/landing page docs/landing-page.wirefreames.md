## 5. Notes

### 5.1 Grid System
- 12-column grid for desktop
- 8-column grid for tablet
- 4-column grid for mobile
- 24px gutters

### 5.2 Spacing
- Section padding: 80px (desktop), 48px (tablet), 32px (mobile)
- Element spacing: 24px base unit
- Line height: 1.5 for body, 1.2 for headings

### 5.3 Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
