# Someday Maybe

**Purpose**: Future possibilities and ideas that aren't active right now.

## File Naming

`YYYY-MM-DD-{title}.md`

## When to Use Someday Maybe

- **Future Projects**: Good ideas but not right now
- **Opportunities**: Potential future directions
- **Learning Goals**: Skills to develop someday
- **Big Ideas**: Visionary concepts needing more thought
- **Low Priority**: Nice to have but not essential

## Template

```markdown
# {Title}
**Added**: {YYYY-MM-DD}
**Category**: {project/idea/learning/opportunity}
**Priority**: {low/medium/high (for future consideration)}

## The Idea
{Clear description of the concept or opportunity}

## Why It's Not Active Now
{Reasons this isn't currently a project - time, resources, priorities}

## What Would Make It Active
{Conditions or triggers that would move this to projects}

## First Step (When Active)
{What would be the very first action if this became a project}

## Related Items
{Connections to current projects, reference materials, etc.}
```

## Review Process

- **Monthly**: Review during monthly review
- **Activate**: Move to `projects/` if timing is right
- **Archive**: Remove if no longer relevant
- **Combine**: Merge similar ideas

## Examples

- "Build mobile app version"
- "Explore European market expansion"
- "Learn advanced blockchain development"
- "Create partnership with exchange platform"
- "Develop AI-powered portfolio recommendations"
