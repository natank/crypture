# Next Actions

**Purpose**: Single physical actions ready to be done, organized by context.

## Contexts

- **@computer**: Actions requiring computer/internet
- **@calls**: Phone calls, video calls, meetings
- **@errands**: Physical tasks outside home/office
- **@home**: Actions at home location
- **@waiting**: Actions blocked by waiting for someone/something

## File Naming

`YYYY-MM-DD-{title}.md`

## Action Template

```markdown
# {Action Title}
**Context**: {@computer/@calls/@errands/@home/@waiting}
**Project**: {Related project if any}
**Due**: {Date if applicable}
**Created**: {YYYY-MM-DD}

## What Needs to Be Done
{Clear, physical next action - should start with a verb}

## Notes
{Any additional context or information}
```

## Action Guidelines

1. **Physical Verbs**: Start with action verbs (Call, Research, Write, Review, etc.)
2. **Specific**: Clear what "done" looks like
3. **Context**: Assign appropriate context for where/when it can be done
4. **Single Step**: If multiple steps needed, it's a project

## Processing

- Review daily/weekly for current context
- Mark complete and archive when done
- Move to waiting-for if blocked
