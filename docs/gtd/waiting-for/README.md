# Waiting For

**Purpose**: Items delegated to others or awaiting external response.

## File Naming

`YYYY-MM-DD-{title}.md`

## Template

```markdown
# Waiting For: {Title}
**From/To**: {Person or organization}
**Waiting Since**: {YYYY-MM-DD}
**Follow Up**: {YYYY-MM-DD} (if applicable)
**Related Project**: {Project name if applicable}

## What I'm Waiting For
{Clear description of what you're waiting to receive/hear}

## Next Action If No Response
{What to do if they don't respond by follow-up date}

## Context
{Why this matters, background information}
```

## Review Process

- **Weekly**: Check for overdue items
- **Follow Up**: Send reminder if past due date
- **Move**: If completed, archive or move to actions
- **Escalate**: If repeatedly delayed, consider alternative approaches

## Common Examples

- API key approval from service provider
- Response from potential partner
- Feedback from beta testers
- Legal document review
- Design asset delivery
- Contractor work completion
