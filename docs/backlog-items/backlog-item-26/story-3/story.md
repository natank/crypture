# Story 3: Navigation & Discovery

## Description
As a user, I want to easily navigate to coin detail pages from my portfolio and market listings, and discover related coins, so that I can explore cryptocurrencies seamlessly.

## Acceptance Criteria
- [ ] "View Details" link/button on portfolio asset rows
- [ ] "View Details" link/button on market coin listings
- [ ] Back navigation to previous context
- [ ] Related coins section (if API data available)

## Technical Tasks
1. Add navigation link to `AssetRow` component
2. Add navigation link to market listing components
3. Implement breadcrumb or back button on `CoinDetailPage`
4. Add related coins section (optional, based on API categories)
